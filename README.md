# Iconic Button

A beautiful, accessible icon button block for WordPress Gutenberg. WCAG 2.1 AA compliant, performance-optimized.

## Description

Create stunning icon buttons with Font Awesome integration. Free version with 32 icons.

### Key Features

- Gutenberg-native blocks
- Fill & Outline styles
- Full accessibility (WCAG AA)
- Automatic dark mode
- RTL & reduced motion support
- No bloat: 1.7KB JS, 15KB CSS

## Installation

1. Download from WordPress.org
2. Upload to `/wp-content/plugins/`
3. Activate
4. Use "Iconic Button" block

## Development

Clone repo, `npm install`, `npm run build`.
